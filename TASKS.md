## 新对话启动流程

1. 读 `CLAUDE.md` → 项目定义、架构、规则、**工作规范**
2. 读 `架构文档.md` → 当前架构、版本记录、开发进度
3. 读本文件 → 找到当前版本未完成的 Task
4. `git log --oneline -5` → 确认最新提交和进度
5. 问用户"从哪个 Task 开始？"；若无未完成 Task，问"下一个版本做什么？"
